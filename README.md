# Fawry E-Commerce Task

This is a simple Java console application built for the Fawry Quantum Internship challenge. It allows customers to add products to a shopping cart and complete the checkout process.  
The system supports different product types, including expirable and shippable items, and handles validations such as stock availability, product expiry, and customer balance.  
Shipping items are processed separately with their total weight.

## Features


- **Flexible Product Types:** Supports various kinds of products, including items that can expire and those that require shipping.

- **Cart Management:** Customers can add available products in desired quantities, and view the full cost including any shipping charges.

- **Secure Checkout:** Ensures the order is valid by checking product stock, expiration status, and the customer’s available balance before proceeding.

- **Shipping Integration:** Gathers all products that require shipping, calculates total weight, and prints a clear summary for delivery handling.

 
## How to Run

- Clone the repository:
`git clone https://github.com/emandeyab/Fawry-Task-Ecommerce.git`
- Open the project in IntelliJ or any Java IDE.
- Run the `Main.java` file.
- The output will show different test cases in the console.
